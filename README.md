# SNP Chatbot Mobile

## Cách dùng
1. Fork repo này về tài khoản GitHub của bạn.
2. Vào tab Actions → bật GitHub Actions.
3. Chạy workflow **Build Android APK (SNP Chatbot)**.
4. Sau khi build xong → tải file `.apk` trong mục Artifacts.

## Sử dụng app
- Khi mở app, nhập URL backend (ví dụ: https://server:8000).
- Gõ câu hỏi và nhận phản hồi.
